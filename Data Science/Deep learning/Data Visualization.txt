seaborn library:
----------------
seaborn is a DATA VISUALIZATION LIBRARY

seaborn is built on top of MATPLOTLIB

what all advantages seaborn provides as compared to matplotlib ?
answer: 
- Better default color palette and layout
- Working with data frames directly

Function to load existing datasets in python notebook
answer: load_dataset()

# CATPLOT:
---------

import seaborn as sns
import pandas as pd
import matplotlib.pyplot as plt
sns.set()

df = sns.load_dataset('tips')
df.head()

sns.catplot(x = 'sex', y= 'total_bill', data = df)
plt.show()

sns.catplot(x='sex', y='total_bill', data=df, jitter = 0.125)
plt.show()

sns.catplot(x='sex', y='total_bill', data=df, jitter = 0.015)
plt.show()

sns.catplot(x='sex', y='total_bill', data=df, kind = 'swarm', hue='smoker')
plt.show()
